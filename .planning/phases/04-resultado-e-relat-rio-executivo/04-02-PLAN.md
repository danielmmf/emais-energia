# PLAN-04: Implementar motor de recomendação textual com próximos passos

## Objective
Implementar motor de recomendação textual que gera recomendações coerentes com os resultados da simulação e sugere próximos passos acionáveis.

## Requirements Addressed
- RES-02: Sistema gera recomendação textual simples com próximos passos

## Tasks
1. Analisar padrões de recomendação baseados na classificação de viabilidade
2. Implementar função que gera resumo textual baseado nos resultados da simulação
3. Implementar função que gera lista de próximos passos baseada na classificação
4. Integrar com ViabilityService para acessar dados de simulação e classificação
5. Criar interface para exibir recomendação textual na tela
6. Garantir que recomendações sejam claras, objetivas e acionáveis
7. Adicionar recomendações de monitoramento para todos os cenários
8. Testar com diferentes combinações de resultados de simulação

## Acceptance Criteria
- [ ] Sistema gera recomendação textual coerente com os resultados da simulação
- [ ] Recomendações consideram tanto payback financeiro quanto fatores estratégicos
- [ ] Próximos passos são específicos e acionáveis para cada classificação
- [ ] Sempre inclui recomendação para consultar especialistas em energia renovável
- [ ] Recomendações variam apropriadamente entre classificações Alta, Média, Baixa, Estratégica e Não recomendada
- [ ] Linguagem é clara e acessível para não técnicos
- [ ] Inclui métricas-chave na recomendação (economia, payback, redução de emissões)

## Dependencies
- ViabilityService (para acessar resultados de simulação e classificação)
- ReportController (para exibir recomendação)
- Formatters existentes (asCurrency, asPercentage)

## Estimated Effort
1.5 dias de desenvolvimento
