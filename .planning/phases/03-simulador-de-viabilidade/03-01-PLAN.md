# PLAN-01: Implement ViabilityService for Economic Calculations

## Objective
Create a centralized service to handle all economic calculations for the viability simulator, implementing requirements SIM-02 through SIM-06.

## Acceptance Criteria
- Service can calculate custo atual anual (gasto_mensal * 12)
- Service can calculate custo verde anual using route-specific factors
- Service can calculate economia anual (custo_atual_anual - custo_verde_anual)
- Service can calculate payback (investimento / economia_anual) when economia anual > 0
- Service can estimate redução de emissões based on route factor
- Service handles both positive and negative economic scenarios
- Service is injectable and testable
- All calculations use mock factors per route as specified in requirements

## Tasks
1. Create ViabilityService.js in app/services/ with AngularJS service structure
2. Implement calculateCustoAnual(gastoMensal) method
3. Implement calculateCustoVerdeAnual(gastoMensal, fatorRota) method
4. Implement calculateEconomiaAnual(custoAtual, custoVerde) method
5. Implement calculatePayback(investimento, economiaAnual) method with validation
6. Implement calculateReducaoEmissoes(gastoMensal, fatorEmissao) method
7. Implement main calculateViabilidade method that orchestrates all calculations
8. Add proper error handling for edge cases (zero division, negative values)
9. Ensure service follows AngularJS dependency injection patterns
10. Write basic unit tests for each calculation method
